# Demo Video Instructions

## 90-Second Demo Recording Guide

This document provides instructions for recording a 90-second video showcasing the Physical AI & Humanoid Robotics Textbook features.

### Demo Flow (90 seconds total)

#### Opening (0-10 seconds)
- Introduce the Physical AI & Humanoid Robotics Textbook
- Mention the key features you'll demonstrate

#### Feature 1: Interactive Learning (10-30 seconds)
- Navigate to Chapter 1: Introduction to Physical AI
- Show the ROS 2 code examples
- Demonstrate the interactive elements (if implemented)

#### Feature 2: RAG Chatbot (30-55 seconds)
- Open the chatbot interface (bottom-right corner)
- Ask a question about Physical AI concepts
- Show how the chatbot provides context-based answers
- Highlight that it's using textbook content for responses

#### Feature 3: Personalization (55-75 seconds)
- Click the "Personalize" button
- Show the user preferences menu
- Demonstrate how content can be customized based on learning preferences

#### Feature 4: Urdu Translation (75-90 seconds)
- Click the language toggle button
- Show the Urdu translation feature
- Demonstrate real-time content translation

### Technical Setup

#### Recording Tools
- Use screen recording software (OBS Studio, CamStudio, or built-in Windows Game Bar)
- Ensure good audio quality if including narration
- Record at 1080p resolution minimum

#### Environment
- Start with a clean browser window
- Close unnecessary tabs and applications
- Ensure good lighting if including facecam

### Demo Script

**Opening (0-10s):**
"Welcome to the Physical AI & Humanoid Robotics Textbook. This interactive learning platform features RAG chatbot assistance, personalization, and Urdu translation."

**Interactive Learning (10-30s):**
"Let's start with Chapter 1 on Physical AI fundamentals. Here you can see code examples for ROS 2 concepts that you can interact with directly."

**RAG Chatbot (30-55s):**
"Now let me show you our RAG chatbot. I'll ask about Physical AI concepts and see how it provides answers based on the textbook content. The chatbot understands the context and provides relevant information."

**Personalization (55-75s):**
"The platform adapts to your learning style. I can set my preferences for difficulty level and learning style to get personalized content recommendations."

**Urdu Translation (75-90s):**
"For Urdu speakers, we provide real-time translation. You can switch to Urdu and access all content in your native language."

### Tips for Recording
- Practice the flow a few times before recording
- Keep transitions smooth and natural
- Narrate what you're doing as you demonstrate
- Keep the pace consistent
- End with a call to action: "Try the Physical AI & Humanoid Robotics Textbook today!"

### Quality Check
- Verify all features work before recording
- Test the chatbot responses beforehand
- Ensure translation functionality works
- Check that personalization options are accessible
- Confirm the video is exactly 90 seconds or less
