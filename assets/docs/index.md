# Course Modules

Welcome to the Physical AI & Humanoid Robotics course modules. This section contains all the learning materials for the course.

## Available Modules

1. [Module 1: ROS 2 Fundamentals](./module1-ros2-fundamentals.md)
   - Learn the fundamentals of Robot Operating System 2, including nodes, topics, services, and actions for building robust robotic applications.

2. [Module 2: Digital Twin (Gazebo & Unity)](./module2-digital-twin.md)
   - Explore digital twin technologies using Gazebo physics simulation and Unity 3D for creating virtual replicas of robotic systems.

3. [Module 3: NVIDIA Isaac AI & Perception](./module3-nvidia-isaac.md)
   - Master NVIDIA Isaac SDK for AI-powered robotics, including perception algorithms and GPU-accelerated computing.

4. [Module 4: Vision-Language-Action (VLA)](./module4-vla.md)
   - Understand Vision-Language-Action models for multimodal robotics applications and human-robot interaction.

5. [Capstone Project: Autonomous Humanoid Robot](./capstone-project.md)
   - Integrate all concepts in a comprehensive project building an autonomous humanoid robot system.

Select any module above to begin your learning journey!
