This page dynamically displays all available modules for the Physical AI & Humanoid Robotics course.

4. **[Module 4: Vision-Language-Action (VLA)](./modules/module4-vla.md)**
   - Understand Vision-Language-Action models for multimodal robotics applications and human-robot interaction.

5. **[Capstone Project: Autonomous Humanoid Robot](./modules/capstone-project.md)**
   - Integrate all concepts in a comprehensive project building an autonomous humanoid robot system.
